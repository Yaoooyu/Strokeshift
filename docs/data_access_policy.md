# Data Access Policy

StrokeShift data are controlled clinical research data. Access is purpose-limited, time-limited, non-transferable, and subject to institutional approval and a signed data use agreement. Public code access does not imply data access. Approved recipients must use institutionally managed storage, least-privilege access controls, encryption, audit logging, and an agreed destruction process.

Patient-level predictions, embeddings, reconstructed reports, image derivatives, and trained weights may remain sensitive and must be reviewed before export.
