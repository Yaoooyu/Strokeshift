# Data Access Request

The StrokeShift dataset cannot be distributed publicly because it contains sensitive, institutionally governed clinical data.

Researchers requesting access must:

1. Contact the corresponding author using an institutional email address.
2. Describe the proposed research purpose, requested data elements, security environment, and intended retention period.
3. Obtain any required local ethics or institutional approvals.
4. Sign the applicable data use agreement before receiving access.
5. Use the data only for the approved purpose and within the approved computing environment.
6. Not attempt re-identification or linkage to external identifiable sources.
7. Not redistribute the data, credentials, patient-level derivatives, reports, images, or trained weights.
8. Delete or return the data when the agreement expires.

Submission of a request does not guarantee access. Approval is subject to institutional review, legal constraints, ethics requirements, and the authors' ability to share the requested data.
